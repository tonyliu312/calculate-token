========================================
Windows打包文件清单
========================================

✅ 已准备好的文件：

1. Python依赖包
   位置: packages/
   大小: 149MB
   用途: 离线安装Python依赖

2. Tokenizer文件
   位置: tokenizers/
   大小: 268MB
   模型数量: 11个
   - Qwen3系列: 7个
   - DeepSeek系列: 4个

3. 源代码文件
   - app.py (Flask应用)
   - calculate_tokens.py (核心逻辑)
   - web/ (Web界面)
   - 其他支持文件

4. 打包脚本
   - build_windows.py
   - package_for_windows.bat
   - TokenCalculator.spec

5. 安装脚本
   - install_offline.bat (离线安装)
   - download_packages.bat (下载依赖)

========================================
需要在Windows上完成
========================================

1. 安装Python 3.8-3.12
2. 安装依赖（在线或离线）
3. 运行打包脚本
4. 测试打包结果

========================================
快速命令
========================================

# 安装依赖（有网络）
pip install -r requirements.txt

# 安装依赖（无网络）
install_offline.bat

# 打包
package_for_windows.bat

# 或
python build_windows.py
