# 直接运行Python项目说明

## ✅ 可以直接运行Python，无需打包！

如果bat脚本有编码问题，完全可以直接运行Python脚本。

## 🚀 快速开始（2步）

### 步骤1：安装依赖

**有网络时**：
```cmd
pip install -r requirements.txt
```

**无网络时**（使用已下载的离线包）：
```cmd
python -m pip install --upgrade pip --no-index --find-links=packages
python -m pip install -r requirements.txt --no-index --find-links=packages
```

### 步骤2：运行应用

**最简单的方式**：
```cmd
python app.py
```

或者使用简化的启动脚本：
```cmd
python run.py
```

或者使用修复后的bat脚本：
```cmd
run_app.bat
```

## 🌐 访问Web界面

运行后，浏览器打开：**http://localhost:5000**

如果5000端口被占用，应用会自动尝试5001端口，查看控制台输出。

## ✅ 验证安装

运行以下命令检查环境是否正确：

```cmd
python -c "import flask; import transformers; print('环境OK')"
```

## 📁 必需的文件

确保以下目录存在：
- ✅ `tokenizers/` - Tokenizer文件（11个模型）
- ✅ `web/` - Web界面文件

## 🔧 常见问题

### Q: 提示找不到模块？

**A**: 安装依赖：
```cmd
pip install -r requirements.txt
```

### Q: 提示找不到tokenizer？

**A**: 检查 `tokenizers/` 目录是否存在且包含模型文件。

### Q: 端口被占用？

**A**: 可以指定端口：
```cmd
set PORT=5001
python app.py
```

### Q: bat脚本乱码？

**A**: 直接使用Python命令，不需要bat脚本：
```cmd
python app.py
```

## 💡 优势

直接运行Python的优势：
- ✅ 无需打包，文件更小
- ✅ 启动更快
- ✅ 更容易调试和更新
- ✅ 不依赖bat脚本编码

## 📝 完整命令示例

```cmd
REM 1. 进入项目目录
cd C:\path\to\calculate-token

REM 2. 安装依赖（如果有网络）
pip install -r requirements.txt

REM 3. 运行应用
python app.py

REM 4. 浏览器访问
start http://localhost:5000
```

## 🎯 总结

**最简单的方式**：直接运行 `python app.py` 即可，无需任何bat脚本！
